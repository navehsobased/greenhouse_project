# 🌱 חממה חכמה אוטונומית - פרויקט גמר

פרויקט גמר - חממה אוטונומית המנהלת את עצמה בזמן אמת.

## 🎓 פרטי הפרויקט
- **מגיש:** נווה אברג'ל
- **מרצה מנחה:** מארק טסליצקי
- **מוסד:** אורט בראודה כרמיאל

## 🌟 תכונות
- בקרה אוטומטית על טמפרטורה, לחות, תאורה והשקייה
- מערכת חכמה לניהול גג מתכוונן
- תצוגת נתונים בזמן אמת על מסך LCD
- קוד פתוח זמין להורדה
